HTML Frame Form

Jan 06, 2018

HTML Frame

cols / rows with walues (number for pxiels or %) to define the frame, no resize to restrict the frame not adjusted by mouse drag
cols/ rows frames can be nested to construct more complext frame

HTML Table

Table tag to create tables, tr tag for table rows, td tag for data inside the row. Nested tds are the table columns from left to right; th tag for table head.

border value indicate the width of the border in pixel, without border attribute or set the value to 0 creates no-border table;

p / table / list can be nested in table tags

Attributes are almost self explanatory, all you need is to code to internalize them

HTML Form

form tag creats input forms, values of type define different types of forms,the values could be checkbox / radio / password and other values url number range Date pickers (date, month, week, time, datetime, datetime-local)search color] for HTML5
select and option tags create drop-down options, “selected” specify the pre-selected option
Create a button
Fields tags display border frames around the form data
The data (Mickey Mouse) will be submitted to demo_form.asp page

To submit successfully the input tag need a “name” attribute with value

The selected checkbox will be submitted to demo_form.asp page
Form to send email
3 added forms in HTML5 (datalist / keygen / output)
Datalist give out predefined values
